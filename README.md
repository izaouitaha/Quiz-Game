# 🧠 Quiz Game using Python & Tkinter

This project is an interactive Quiz Application built using Python and Tkinter.

The application displays multiple-choice questions loaded from a JSON file. Users can select answers, receive instant feedback, and track their score in real time.

---

## 🚀 Features

- Graphical User Interface (GUI) using Tkinter
- Questions loaded from an external JSON file
- Multiple-choice answers (buttons)
- Instant feedback (correct / wrong)
- Score tracking system
- Restart functionality
- Simple and clean design
